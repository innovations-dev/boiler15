## Key Features

- **Authentication**: Built-in auth system with Better-Auth
- **Database**: Drizzle ORM with Turso (SQLite)
- **Email**: React Email with Resend
- **UI Components**: Shadcn UI with Tailwind CSS
- **Type Safety**: End-to-end TypeScript with Zod validation
- **Documentation**: Built-in MDX support

## Core Principles

1. **Simplicity**: We prioritize simple, maintainable solutions over complex implementations
2. **Modern Patterns**: Utilizing React Server Components and other modern React patterns
3. **Type Safety**: Full TypeScript support throughout the application
4. **Performance**: Optimized for Core Web Vitals and user experience
5. **Developer Experience**: Comprehensive tooling and documentation

## Project Structure

```text
├── app/ # Next.js app directory
├── components/ # Reusable components
├── content/ # MDX content and documentation
├── lib/ # Utility functions and shared code
├── styles/ # Global styles and Tailwind config
└── types/ # TypeScript type definitions
```

## Prerequisites

Before you begin, make sure you have the following installed:

- Node.js 18.17 or later
- pnpm 8.0 or later
- Git
